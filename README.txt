Title: C195 Scheduler Program
Purpose: This application manages customers and their respective appointments backed up by
a MYSQL database. Users must login to access the create/read/update/delete functions of the
application.

--------------------------------------------------------------------

How to run the program:
1) Ensure all database and Java requirements listed above are installed and configured.
2) Open the project in IntelliJ IDEA.
3) Connect the MySQL database.
4) Setup the Run/Debug Configurations to run the Scheduler application
5) Click the green "play" button in the top right of IntelliJ

--------------------------------------------------------------------

Additional Report Description:
The additional report will provide a combo box for the user to select a customer. The customer
selected will have their respective appointment ID's and length of each appointment populated
in the text area (directly to the right) of the combo box. The lengths will display the
appointment lengths in hours and minutes.
